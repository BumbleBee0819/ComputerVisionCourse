Assignment 3 Overview: Laplacian blending

The goal of this project is to use image-blending techniques to seamlessly merge information from multiple images.

To conduct Laplacian blending, I first got the Gaussian pyramids by convolving the original image with a Gaussian filter and then downsampling. Then the Laplacian pyramids were generated by: Laplacian [level (a-1)]=Gaussian [level (a-1)]+ interpolate (Gaussian [level a]). After I got the Laplacian pyramids for both the target and the source images, I first interpolated all images to Level 0 size, then for each Laplacian level, I blended them using: blended image= Gaussian (mask)*Laplacian(source)+(1- Gaussian(mask))* Laplacian(target).

There are two tips in this method: firstly, we have to padding the image so that the image size is 2**n; secondly, the mask must be normalized before blending, which means that 0=0 (black) and 1=255 (white).
